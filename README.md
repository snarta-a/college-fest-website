# 🎓 College Fest 2026 Website

A modern, interactive multi-section website designed for a college fest event. Built using **HTML, CSS, and JavaScript**, this project showcases event details, schedule, and a fully functional registration system.

---

## 🚀 Features

* 🎨 Beautiful UI with glassmorphism design
* 📅 Multiple event sections (Singing, Coding, Dance, etc.)
* ⏳ Live countdown timer
* 📝 Interactive registration form with validation
* 📊 Event schedule with tabs
* 🎯 Smooth scrolling & animations
* 🔔 Toast notifications & user feedback
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* **HTML5** – Structure (header, nav, article, footer)
* **CSS3** – Styling (Flexbox, Grid, animations, colors, spacing)
* **JavaScript** – Interactivity (form validation, events, DOM manipulation)
